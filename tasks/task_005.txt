# Title: Investment Performance Analysis
# Status: pending
# Dependencies: 2, 4
# Priority: high
# Description: Add tools for analyzing investment performance across different time periods.
# Details:
Create a Performance Analysis sheet with the following features:

1. Performance Overview section:
   - Total return, return percentage, annualized return
   - Volatility and Sharpe ratio calculations

2. Time Period Analysis section:
   - Performance metrics for different time frames (1D, 1W, 1M, 3M, 6M, YTD, 1Y, All)
   - Comparison chart of returns by time period

3. Top Performers section:
   - List of best performing holdings with return metrics
   - Sortable by different metrics

4. Bottom Performers section:
   - List of worst performing holdings with return metrics

Implement helper functions to calculate:
- Performance metrics based on historical data
- Time-weighted returns
- Performance by timeframe
- Individual holding performance with cost basis calculation

# Test Strategy:
1. Test calculations with known portfolio data and verify results
2. Check edge cases like newly added positions or recently sold positions
3. Verify time period calculations with different date ranges
4. Test with various portfolio compositions
5. Verify that performance metrics match expected financial calculations
